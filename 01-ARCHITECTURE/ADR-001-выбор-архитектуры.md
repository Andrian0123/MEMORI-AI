# ADR-001: Выбор архитектуры

## Контекст

Требуется выбрать архитектуру для мобильного приложения «Смета-А», которое будет работать с 3D-сканером, BIM-редактором и сметным расчётом.

## Решение

Выбрана **Clean Architecture** с разделением на слои:
- Presentation (UI)
- Domain (бизнес-логика)
- Data (источники данных)

## Последствия

### Позитивные
- Тестируемость
- Независимость от фреймворков
- Возможность масштабирования

### Отрицательные
- Сложность начальной настройки
- Больше кода для простых функций
