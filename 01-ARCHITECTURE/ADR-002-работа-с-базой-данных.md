# ADR-002: Работа с базой данных

## Контекст

Требуется выбрать СУБД для локального хранения смет, BIM-моделей и настроек пользователя.

## Решение

**Room** как основной источник данных для структурированных данных + **Realm** для временных 3D-данных сканера.

## Последствия

### Позитивные
- Room: типобезопасность, миграции, производительность
- Realm: хранение больших файлов, синхронизация

### Отрицательные
- Две СУБД усложнят отладку
